## 셔틀버스

- [출처](https://programmers.co.kr/learn/courses/30/lessons/17678)

### 입력

- n - 버스 정차 횟수
- t - 버스 정차 간격
- m - 버스 탑승 가능 승객
- timetable - 사람들 나오는 시간 배열

### 출력

- 콘이 무사히 셔틀을 타고 사무실로 갈 수 있는 제일 늦은 도착 시각을 출력한다. 도착 시각은 HH:MM 형식이며, 00:00에서 23:59 사이의 값이 될 수 있다.

### 아이디어

- 시뮬레이션 문제. 구현력을 중요시하는 문제였다.
