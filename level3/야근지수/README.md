## 야근지수

- [출처](https://programmers.co.kr/learn/courses/30/lessons/12927)

### 입력

- n - 퇴근까지 남은 시간
- works - 남은 작업량 배열

### 출력

- 야근 피로도를 최소화한 값

### 아이디어

1. 전체 작업량이 남은 시간보다 작거나 같을 때, 얼리 리턴
2. works 중 최대 작업량 구하기
3. 남은 시간이 0이 될 때까지 loop
   1. works\[i\]가 최대 작업량과 같다면 해당 작업량과 남은 시간 1씩 차감
   2. 최대 작업량 1 차감
